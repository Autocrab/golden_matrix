# golden_matrix — обзор продукта

## Что это

**golden_matrix** — Flutter-пакет для **matrix-based visual regression testing**.

Разработчик описывает одну декларацию матрицы, а пакет сам разворачивает её в набор тестовых кейсов, golden-файлов, diff-артефактов и отчётов.

## Какую боль решает

Классические golden tests во Flutter проверяют **один конкретный кейс**. При добавлении тем, локалей, device sizes и состояний начинается копипаст и комбинаторный взрыв.

Типичные баги, которые пропускаются:
- на немецком/арабском строка не помещается;
- на `textScaleFactor = 2.0` разъезжается layout;
- в dark mode пропадает контраст;
- на маленьком девайсе overflow;
- в RTL ломаются отступы;
- `loading`/`error`/`empty` состояния никто не проверял.

## Главная идея

> Один сценарий → много сред → предсказуемые артефакты → понятный отчёт → контролируемый baseline workflow.

## Целевая аудитория

- Команды 3–15 Flutter-разработчиков
- Приложения с несколькими локалями, dark/light theme
- Проекты с большим количеством форм и состояний
- White-label приложения
- Design system teams

## Ценность

| Для кого | Что даёт |
|---|---|
| Разработчики | Меньше ручного кода и копипаста, проще покрыть edge cases |
| QA / команда | Прозрачная визуальная регрессия, HTML/JSON отчёты |
| Дизайн-системы | Массовая проверка компонентов через темы, text scale, устройства |
| Продуктовые команды | Меньше багов на локалях, в dark mode, accessibility |
